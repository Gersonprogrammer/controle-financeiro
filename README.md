# 💰 Controle Financeiro

Aplicação web para controle de finanças pessoais, desenvolvida como projeto de portfólio para praticar desenvolvimento web, integração com APIs e banco de dados PostgreSQL utilizando Supabase.

---

## 🎯 Objetivo

Permitir que o usuário controle suas receitas e despesas de forma simples, rápida e intuitiva.

---

## 🚀 Tecnologias
HTML5
CSS3
JavaScript
Git
GitHub
Firebase ou Supabase (conforme a evolução do projeto)

---

## 📋 Funcionalidades (Versão 1.0)

Dashboard financeiro
Controle de receitas
Controle de despesas
Categorias
Relatórios
Gráficos
Metas financeiras
Contas bancárias
Cartões de crédito
Orçamento mensal
Backup em nuvem
Sincronização entre dispositivos
Inteligência Artificial para análise financeira (planejada)
